# Ludo Game (Chińczyk) – C++ / wxWidgets

Desktop implementation of the classic **Ludo (Chińczyk)** board game written in C++ using wxWidgets for GUI.

## Features

* 4-player turn-based gameplay
* Full Ludo rules implementation
* Dice rolling system
* Piece movement & collision logic
* Win condition detection
* GUI built with wxWidgets

## Technologies

* C++
* wxWidgets
* Code::Blocks

## Project Structure

* `GameDialog.*` – main game logic and board
* `Gra.*` – game mechanics
* `Zasady.*` – rules dialog
* `Projekt_chinczykMain.*` – start menu
* `assets/` – images and game graphics

## How to Run

Open `Projekt_chinczyk.cbp` in Code::Blocks
Build and run the project.

Make sure the `assets/` folder is in the same directory as the executable.
